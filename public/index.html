JSON Format (.json)

Must be an array of objects. Supports multiple images. The correct_option must match one of your choices exactly. 

[
    {
        "question_text": "What is 2 + 2?",
        "correct_option": "4",
        "options": ["3", "4", "5"],
        "images": ["/images/math.png"]
    }
]

CSV Format (.csv)

Columns must follow this exact order:
Question, Correct Choice, Image Path, Option A, Option B...

*Wrap fields in double quotes if they contain commas. Leave Image Path blank if not needed. 

QuestionText,CorrectOption,ImageUrl,OptionA,OptionB,OptionC
What is 2+2?,4,,3,4,5
"Hello, World!",World,/img.png,World,Earth,Space
